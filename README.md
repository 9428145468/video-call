# Typescript-react-webrtc-1-1

## Features
- 1:1 communication (peer to peer)
- React with Typescript

### Node.js and React.js
You need to install Node.js
```
cd Typescript-react-webrtc-1-1
npm install
node server.js
cd web
npm install
npm start
```
